## 🎯 Project Levels & Tasks

### 🌱 Level 1: The Foundations
* **Task 1: Data Preprocessing for Machine Learning**
  * Handled missing values, encoded categorical variables, and performed feature scaling (Standardization).
* **Task 2: House Price Prediction (Linear Regression)**
  * Built a predictive model to estimate house prices based on various structural and locational features. Evaluated using Mean Squared Error (MSE) and R-squared metrics.

### 🌿 Level 2: Intermediate Classification
* **Task 1: Customer Churn Prediction (Logistic Regression)**
  * Developed a binary classification model to predict whether a customer will leave a telecommunications company. Visualized performance using ROC curves and calculated odds ratios for feature importance.
* **Task 2: Iris Flower Classification (Decision Trees)**
  * Implemented a Decision Tree classifier to identify iris species. Successfully plotted the decision tree logic to visually interpret how the algorithm makes splitting decisions with 100% accuracy.

### 🌳 Level 3: Advanced ML & Deep Learning
* **Task 1: Advanced Churn Prediction (Random Forest)**
  * Upgraded the Level 2 churn model using a Random Forest Classifier with hyperparameter tuning via `GridSearchCV`. Improved accuracy to >95% and extracted critical feature importances.
* **Task 2: Sentiment Analysis (Neural Networks & NLP)**
  * Built a Feed-Forward Neural Network using TensorFlow/Keras to analyze textual data. Used TF-IDF vectorization to convert text into numerical format, successfully classifying user sentiments into Positive, Negative, and Neutral categories.
